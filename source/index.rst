=====
AROMA
=====

AROMA (Advanced Reduced Order Modelling Applications) is a Python
package for developing fast Reduced Basis Methods. It is currently
tightly integrated with `Nutils <https://nutils.org>`_.

The journey to start using AROMA begins with an understanding of
:doc:`case`.
